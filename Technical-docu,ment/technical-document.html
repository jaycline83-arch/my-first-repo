HTML Basics

HTML is the structure of a webpage.

It uses tags like <p> for paragraphs and <h1> for headings.

Every page starts with <!DOCTYPE html> and <html> tags.

Inside <body> you put the content that shows on the page.

<p>This is a paragraph</p>
  • <h1> to <h6> for headings
  • <p> for text paragraphs
  • <a> for links
CSS Basics

CSS adds style to your HTML.

You can change colors, fonts, and layout.

CSS rules look like this:

p { color: blue; font-size: 16px; }

Selectors choose elements, properties change how they look.

  • color changes text color
  • background-color changes background
  • margin and padding add space
Responsive Design

Responsive design means your page looks good on all devices.

You can use percentages or relative units like em and rem.

Media queries let you change styles for different screen sizes.

@media (max-width: 600px) { body { font-size: 14px; } }

This example makes text smaller on small screens.

  • Use flexible layouts
  • Test on phone and computer
  • Keep it simple
